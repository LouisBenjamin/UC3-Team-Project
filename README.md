

# SOEN 341 - Team UC3

## Objective

Develop a twitter-like software/blog/web application.

## Project Description

Online social networking service on which users post and interact with messages.

## Core Features

* Posting messages or "tweets"
* "Like" a message
* Follow users

## Potential Features

* Reshare posts
* See generated news
* Upload Pictures
* Comment on "tweet"
